# نظام المحاسبة المتكامل - شركة تواريدات

نظام محاسبي متكامل مبني بـ Django + PostgreSQL متوافق مع القانون المصري.

## المميزات

### 1. الحسابات العامة
- شجرة الحسابات حسب المعايير المصرية
- دليل الحسابات الكامل
- القيود المحاسبية (عامة، مشتريات، مبيعات، تحصيل، دفع، إهلاك، رواتب)
- ميزان المراجعة
- ترحيل تلقائي للقيود

### 2. المشتريات
- إدارة الموردين (بيانات، أرصدة، حد ائتمان)
- المنتجات والتصنيفات
- فواتير مشتريات (مؤثرة/غير مؤثرة في الضرائب)
- ربح الشراء والبيع
- قيد محاسبي تلقائي

### 3. المبيعات
- إدارة العملاء (بيانات، أرصدة، حد ائتمان)
- فواتير مبيعات مع نسبة الربح
- الفاتورة الضريبية المصرية
- القيد المحاسبي التلقائي (إيرادات + تكلفة + VAT)

### 4. الضرائب (القانون المصري)
- **ضريبة القيمة المضافة (VAT 14%)**
  - Input VAT (محصلة من المشتريات)
  - Output VAT (مستحقة على المبيعات)
  - إقرار VAT الشهري/الربعي
- **ضريبة الخصم والتحصيل (0.5%)**
  - على المشتريات من غير المسجلين
  - تقرير تفصيلي
- **الفاتورة الإلكترونية (منظومة فاتورة — ETA)**
  - ربط فواتير المبيعات بمنظومة الفاتورة الإلكترونية لمصلحة الضرائب المصرية
  - إرسال المستندات عبر API (`api.invoicing.eta.gov.eg`)
  - متابعة حالة المستند (صالح / مرفوض) واستلام UUID والرقم الطويل للتحقق
  - دعم بيئتي Sandbox والإنتاج
  - تخزين شهادة التوقيع الرقمي (PKI) للتوقيع على المستندات

### 5. البنوك والخزائن
- إدارة البنوك (حسابات، أرصدة)
- المعاملات البنكية (إيداع، سحب، تحويل، شيكات)
- الخزائن النقدية
- سجل المعاملات

### 6. الأصول الثابتة والإهلاك
- تصنيفات الأصول
- الأصول (أراضي، مباني، آلات، سيارات، أثاث)
- طرق الإهلاك (القسط الثابت / المتناقص)
- قيود الإهلاك التلقائية
- الجدول الدفتري

### 7. الموارد البشرية
- الموظفين (بيانات شخصية، وظيفية)
- الأقسام
- الحضور والانصراف
- الرواتب (أساسي، بدلات، إضافي، مكافآت)
- الخصومات (تأمين اجتماعي 11%، ضريبة الدخل)
- القيد المحاسبي التلقائي

### 8. التقارير
- **قائمة الدخل** (إيرادات - تكلفة - مصروفات - ضرائب)
- **الميزانية العمومية** (أصول - خصوم - حقوق ملكية)
- **ميزان المراجعة**
- **إقرار VAT**
- **تقرير الخصم والتحصيل**
- **تقرير نسب الربح** (للمنتجات)
- **تقرير الموردين** (تفصيلي)
- **تقرير العملاء** (تفصيلي)
- **جدول الأصول والإهلاك**
- **تقرير الرواتب**

## التثبيت

### المتطلبات
- Python 3.10+ (يجب ضعف "Add Python to PATH" أثناء التثبيت)
- لا حاجة لـ PostgreSQL — يستخدم SQLite (قاعدة بيانات ملفية)

---

### الطريقة السريعة (التثبيت التلقائي)

1. **فك الضغط** عن المجلد المضغوط على الجهاز الجديد
2. **افتح** المجلد واضغط مرتين على `install.bat`
3. **انتظر** حتى تكتمل الخطوات (8 خطوات)
4. **افتح** المتصفح على `http://127.0.0.1:8000`
5. سجّل الدخول بـ:
   - المستخدم: `admin`
   - كلمة المرور: `admin123`

> **ملاحظة**: ملف `install.bat` يقوم بـ:
> - التحقق من تثبيت Python
> - إنشاء بيئة افتراضية
> - تثبيت المكتبات من `requirements.txt`
> - إنشاء ملف `.env` بـ SECRET_KEY جديد
> - تشغيل migrations على قاعدة البيانات المرفقة
> - إنشاء حساب المدير

---

### الطريقة اليدوية (للمتقدمين)

```bash
# 1. فك الضغط عن المجلد
# 2. افتح Terminal في المجلد

# 3. إنشاء بيئة افتراضية
python -m venv venv
venv\Scripts\activate        # Windows
# source venv/bin/activate   # Linux/Mac

# 4. تثبيت المتطلبات
pip install -r requirements.txt

# 5. إعداد ملف البيئة (ملاحظة: install.bat يفعل هذا تلقائياً)
copy .env.example .env

# 6. تطبيق migrations
python manage.py migrate --no-input

# 7. إنشاء مستخدم المدير
python manage.py create_admin --username admin --password admin123

# 8. تشغيل السيرفر
python manage.py runserver 0.0.0.0:8000
```

---

### للتشغيل بعد التثبيت

| الملف | الوظيفة |
|---|---|
| `install.bat` | التثبيت الأولي (مرة واحدة فقط) |
| `start.bat` | تشغيل السيرفر |
| `stop.bat` | إيقاف السيرفر |

---

### للتشغيل على أجهزة أخرى في الشبكة

1. افتح `start.bat`
2. افتح المتصفح على الجهاز الآخر: `http://IP-الجهاز:8000`
3. للسماح بالاتصال الخارجي، عدّل `.env`:
   ```
   DJANGO_ALLOWED_HOSTS=127.0.0.1,localhost,192.168.1.31
   ```

---

### ملاحظات مهمة

- **قاعدة البيانات**: `db.sqlite3` مرفقة مع جميع البيانات والمخططات — لا تحتاج إعداد PostgreSQL
- **التراخيص**: مفتوح المصدر — لا حاجة لتفعيل
- **النسخ الاحتياطي**: مجلد `backups/` يحتوي نسخ احتياطية تلقائية
- **التحديث**: استبدل الملفات旧 بجديدة ثم شغّل `install.bat` (سيحافظ على قاعدة البيانات)
- **macOS / Linux**: استخدم `setup.sh` بدلاً من `install.bat`
