دليل الإعداد
الجهاز الأول (Host):
  1. اترك نوع الجهاز كـ "مستقل" أو اختر "Host"
  2. تأكد أن السيرفر يعمل على Port 8000
  3. افتح الـ Port في الفايروول
  4. اعطِ الجهاز الآخر مفتاح الـ API
الجهاز الثاني (Client):
  1. ادخل عنوان IP للـ Host
  2. ادخل Port الـ Host (8000)
  3. ادخل مفتاح الـ API من الـ Host
  4. اضغط "اختبار الاتصال"
  5. اضغط "مزامنة الآن"
ملاحظات مهمة:
  • يجب أن يكون الجهازين على نفس الشبكة (LAN) أو متوصلين عبر الإنترنت
  • للإنترنت: افتح Port 800-forward على الراوتر
  • مفتاح الـ API هو كلمة سر الربط بين الجهازين
  • البيانات تُرسل كـ JSON مشفر
الأمان
تشفير الاتصال:

جميع البيانات تُرسل عبر HTTP. للإنتاج، يُنصح باستخدام HTTPS.

مفتاح عشوائي من 64 حرف يُستخدم للمصادقة بين الأجهزة.

  • صفحة الإعدادات تتطلب تسجيل دخول
  • المزامنة التلقائية يمكن تفعيلها/تعطيلها
