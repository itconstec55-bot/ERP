2.5 تكاليف الإنتاج

المسار: إنتاج الخرسانة → تكلفة المتر المكعب

البندالتفاصيل
الغرضحساب التكلفة الفعلية لكل متر مكعب من الخرسانة
المعادلةالتكلفة = (تكلفة الأسمنت + تكلفة الرمل + تكلفة الحصى + تكلفة المواد المضافة + تكلفة النقل) ÷ الكمية

التكاليف المحسوبة تلقائياً:

  • تكلفة الأسمنت = (الكمية المستهلكة × سعر الوحدة)
  • تكلفة العمالة = (عدد العمال × الأجر اليومي)

5اعتماد الفاتورة
6ترحيل الفاتورة (إنشاء القيد المحاسبي + حركة المخزون)

الحسابمديندائن

11.1 FIFO Costing — تكلفة المخزون بطريقة الوارد أولاً

الغرض: حساب تكلفة المخزون بدقة باستخدام طريقة FIFO (First In, First Out) حيث تُحتسب تكلفة البضاعة المباعة بتكلفة أقدم طبقات المخزون أولاً.

المفهومالشرح
طبقة مخزونكل دفعة واردة تُسجّل كطبقة مستقلة بسعرها
FIFOعند البيع، تُخصم الكمية من أقدم طبقة أولاً
التكلفةتكلفة البضاعة المباعة = سعر أقدم طبقة
الرصيدرصيد المخزون = مجموع طبقات المخزون المتبقية

المسار: يتم تلقائياً عند ترحيل فاتورة مبيعات أو إذن استلام.

مهم: FIFO يضمن دقة التكلفة ويناسب صناعة الخرسانة حيث تتغير أسعار المواد الخام باستمرار.
